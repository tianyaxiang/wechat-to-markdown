How to save as favicon.ico:

  1. Use an online converter like Convertio or favicon.io
  2. Upload the SVG file (wechat-markdown-icon.svg)
  3. Download the converted .ico file
  4. Rename it to favicon.ico
  5. Replace the existing favicon.ico in the public folder

Alternatively, you can use a tool like ImageMagick if installed:

convert -background none -density 256x256 wechat-markdown-icon.svg -define icon:auto-resize=16,32,48,64 favicon.ico
